# CSS选择器相关整理

## 主要内容包括：

#### 常用选择器

```

  1. 通配符选择器 ( * )
  2. id选择器（\#id）
  3. 类选择器 （.className）
  4. 元素选择器（E）
  5. 后代选择器（E F）
  6. 子元素选择器（E>F）
  7. 相邻兄弟元素选择器(E+F)
  8. 群组选择器（selector1，selector2，...,selectorN）

  ```
